# 多模态图像检索系统设计

## 项目概述

结合 YOLO 目标检测和 ChromaDB 向量数据库，构建支持文本搜索和图像搜索的多模态检索系统。

## 功能需求

1. **YOLO检测 + 元数据存储**
   - 输入图片，使用 YOLO 检测物体类别、数量、位置
   - 将检测结果作为元数据存入 ChromaDB

2. **CLIP向量搜索**
   - 使用 CLIP 模型提取图片特征向量
   - 存入 ChromaDB，支持相似图片搜索

3. **搜索与展示**
   - 文本搜索：按物体类别搜索图片
   - 向量搜索：上传图片找相似图片
   - 图片标注显示 + 终端详细信息打印

## 技术架构

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  输入图片   │───▶│  YOLO检测   │───▶│ 提取元数据  │
└─────────────┘    └─────────────┘    └──────┬──────┘
                                              │
                                              ▼
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  显示结果   │◀───│ ChromaDB    │◀───│ 存入向量    │
└─────────────┘    └─────────────┘    └─────────────┘
```
